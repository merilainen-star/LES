# Edge pre-version install instructions

Package to share: `edge-extension-1.30-test.zip`

Current test version: `1.30`

1. Download or copy `edge-extension-1.30-test.zip`.
2. Extract the zip to a normal folder, for example `LekolarEnhancer-Edge-test`.
3. Open Microsoft Edge and go to `edge://extensions/`.
4. Turn on **Developer mode**.
5. Click **Load unpacked**.
6. Select the extracted folder that contains `manifest.json`.
7. Confirm that **Lekolar Enhancer** appears in the extension list and shows version `1.30`.

Keep the extracted folder in place while testing. Edge loads the extension from that folder, so deleting or moving it will break the sideloaded extension.
