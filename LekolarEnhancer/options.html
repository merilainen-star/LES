General

Top-level switches and which Lekolar sites the extension activates on.

Features

Price simulation

Adjust displayed prices by a percent for discount or markup testing. Active simulated prices are highlighted and labeled on the page.

Active on

Uncheck a country to turn the extension off on that domain only.

SharePoint access

Required for extended features (price sorting, Swedish reference lookups). Sign in to SharePoint, then re-check.

Status: Unknown

Beta features

Show experimental features (currently: AI Search). Off by default — these need API keys and may change without notice.

External services

AI Search sends the typed query to the AI provider you select. Swedish reference translation sends source text to MyMemory. These features stay off until you allow external data transmission.

External services are off.

Copy format

Each format slot is bound to a modifier-key combo on the copy buttons. Drag tokens to reorder, click ✕ to delete, and use “+ Token” / “+ Text” to add. Live preview updates as you go.

Omnibox search

Type l + space in the address bar. Pick a default country and customize the base search URLs.

Default country

Used when you don't prefix with fi: / se: / no: / dk:.

AI Search BETA

Type a free-text query in Finnish. The chosen provider extracts facets locally with your own API key — keys are encrypted with a non-exportable key in your browser keystore and never leave your machine except to call the provider you pick.

⚠ No browser storage is immune to local malware. Set a low monthly spending cap on each provider account.
